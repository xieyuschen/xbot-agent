# CLI 参考

xbot CLI 终端界面完整参考。

## 键盘快捷键

### 全局

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+K` | 打开命令面板（模糊搜索所有命令） |
| `Ctrl+C` | 取消当前生成 / 中断 |
| `Ctrl+N` | 打开统一 LLM 面板——切换模型 + 管理订阅合一（`订阅名 · 模型名`，跨订阅，按会话，持久化。面板内：`↑↓` 导航，`Enter` 选择/启停，`E` 编辑，`D` 禁用/删除，`N` 添加模型，`S` 显示全部噪声模型，`/` 过滤，`Esc` 关闭） |
| `Ctrl+T` | 打开会话面板 |

### 输入

| 快捷键 | 功能 |
|--------|------|
| `Enter` | 发送消息 |
| `Shift+Enter` | 换行 |
| `↑` / `↓` | 浏览输入历史 |
| `Tab` | 自动补全 |

### 导航

| 快捷键 | 功能 |
|--------|------|
| `/` | 开始输入 slash 命令 |
| `Esc` | 关闭面板 / 取消操作 |
| 鼠标点击 | 点击侧边栏切换会话，滚动消息，点击设置 |

## 鼠标支持

TUI 支持完整的鼠标交互：

- **侧边栏：** 点击任意会话切换，滚动查看所有会话
- **消息：** 滚动浏览对话历史
- **设置：** 点击设置项，使用下拉菜单，切换开关
- **面板：** 点击结果选择
- **进度面板：** 点击展开/折叠子 Agent 详情

## 主题

通过 `Ctrl+K → Theme` 或 `/palette theme` 切换主题。xbot 包含 9 种内置主题，
支持自定义主题（JSON 文件放在 `~/.xbot/themes/`）。

{{< hint type=tip >}}
Agent 可以为你切换主题——只需说 "切换深色主题"，它就会使用 `tui_control` 工具。
{{< /hint >}}

## 模型层

xbot 使用三个模型层应对不同复杂度的任务。通过 `/settings` 配置：

| 层 | 使用场景 |
|------|----------|
| **Vanguard** | 最强推理——复杂任务、架构决策 |
| **Balance** | 均衡——通用工作 |
| **Swift** | 快速/小型——快速查找、简单操作 |

子 Agent 自动选择合适的层。未配置的层自动回退：vanguard → balance → swift。
